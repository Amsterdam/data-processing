load_xls_to_postgres¶

Upload xls files as separate tables into PostgreSQL using Pandas to_sql.

usage: load_xls_to_postgres [-h] datadir config_path dbconfig

Positional Arguments¶

datadir Local data directory, for example: projectdir/data
config_path Location of the config.ini file: for example: /auth/config.ini
dbconfig config.ini name of db settings: dev or docker
