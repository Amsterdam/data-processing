load_wfs_to_postgres¶

Upload gebieden into PostgreSQL from the WFS service of api.data.amsterdam.nl with use of ogr2ogr. Example to run in development: load_wfs_to_postgres config.ini dev

usage: load_wfs_to_postgres [-h] config_path db_config

Positional Arguments¶

config_path Type the relative path + name of the config file, for example: auth/config.ini
db_config Type ‘dev’ or ‘docker’ to setup the proper port and ip settings
